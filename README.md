# Fake News Detection Intelligence System-

AI Fake News Intelligence System is a Machine Learning-based web application that detects whether a news article is Real or Fake.

The system uses TF-IDF for text feature extraction and Logistic Regression for classification. It provides prediction results along with confidence score, fake/real probability, and risk level.

Additionally, the project includes Explainable AI, which highlights the most influential words responsible for the prediction, making the system transparent and user-friendly.

The frontend is built using Streamlit with a modern and interactive UI design
